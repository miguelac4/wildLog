# Build Commands for Staging and Production

## Staging
To build the project for staging, use the following commands:

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Build the project for staging
npm run build -- --mode staging-root
```

## Production
To build the project for production, use the following commands:

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Build the project for production
npm run build -- --mode production-root
```
